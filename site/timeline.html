主线剧情时间线

按"剧情篇章顺序"组织，覆盖成长线、家族线、反魂族线三条主线。

一、总览：主线推进的三条线

1. 成长线 萧炎从乌坦城“斗之气三段”的废柴少年重新起步，在药老指导下重修斗气、学习炼药、吞噬异火、进化焚诀，最终由地方少年成长为斗气大陆顶峰的炎帝。 2. 家族线 萧炎的个人成长背后，是没落萧族与远古八族的旧怨。萧族因萧玄冲击斗帝失败而血脉枯竭，魂族暗中夺取陀舍古帝玉并清洗远古族群，萧炎最终以斗帝身份复兴萧族。 3. 反魂族线 前中期敌人看似是纳兰嫣然、云岚宗、韩枫、魂殿护法、各地强者；后期逐渐收束到魂殿与魂族。魂殿抓捕灵魂、追查古玉、谋夺异火和丹药，最终服务于魂天帝突破斗帝的大局。

二、主线时间线分篇整理

乌坦城废柴与退婚篇
主要地点: 乌坦城、萧家、米特尔拍卖场。 核心人物: 萧炎、萧战、萧薰儿、纳兰嫣然、药老、雅妃、萧宁、萧媚。 萧炎状态: 斗之气低谷重新起步，尚未进入真正大陆舞台。 获得资源: 药老、骨炎戒、筑基灵液、焚诀线索、炼药入门。 剧情意义: 这是萧炎人格与主线目标的起点。“莫欺少年穷”不是单纯口号，而是他后续所有修炼、复仇、自证与守护的精神... 漫画表现重点: 漫画通常强化退婚冲突、萧炎被羞辱后的爆发情绪、药老登场的转折感。

主线事件：

1. 萧炎少年时期曾是萧家天才，后因药老寄居戒指吸收斗气，境界连续倒退。

2. 纳兰嫣然随云岚宗来萧家退婚，萧炎立下“三年之约”。

3. 药老苏醒，解释萧炎斗气消失的原因，正式收萧炎为徒。

4. 萧炎重修，开始炼制筑基灵液，借米特尔拍卖场换取资源。

魔兽山脉修炼篇
主要地点: 魔兽山脉、青山镇、佣兵活动区域。 核心人物: 萧炎、药老、小医仙、云韵、紫晶翼狮王、狼头佣兵团。 萧炎状态: 斗者至斗师阶段快速成长。 获得资源: 飞行斗技线索、药材、毒经关联、小医仙羁绊、云韵羁绊。 剧情意义: 这一篇完成萧炎从“家族子弟”到“独立修炼者”的身份转换，也埋下小医仙、云韵两条重要人物线。

主线事件：

1. 萧炎离开乌坦城进入魔兽山脉修炼。

2. 结识小医仙，获得毒经、宝藏、飞行斗技等机缘。

3. 与云韵相遇，卷入紫晶翼狮王相关事件。

4. 通过实战磨炼，萧炎从家族少年转向真正的冒险者。

塔戈尔沙漠与青莲地心火篇
主要地点: 塔戈尔大沙漠、石漠城、蛇人族领地。 核心人物: 萧炎、药老、美杜莎女王/彩鳞、青鳞、海波东、萧鼎、萧厉、古河、云韵。 萧炎状态: 大斗师阶段，初步掌握异火力量。 获得资源: 青莲地心火、海波东盟友、七彩吞天蟒/彩鳞羁绊、青鳞人物线。 剧情意义: 青莲地心火是萧炎真正意义上的第一朵异火，标志焚诀从潜力功法转入可成长轨道。

主线事件：

1. 萧炎前往塔戈尔沙漠寻找异火。

2. 与大哥萧鼎、二哥萧厉重逢。

3. 结识青鳞，青鳞的碧蛇三花瞳设定出现。

4. 海波东登场，破除封印成为萧炎早期重要盟友。

5. 蛇人族、美杜莎女王、古河一行围绕青莲地心火发生争夺。

6. 美杜莎借青莲地心火进化，意外形成七彩吞天蟒状态。

7. 萧炎最终取得青莲地心火，并推动焚诀第一次核心进化。

关键战斗：沙漠异火争夺、美杜莎进化、萧炎逃生。
帝都炼药与三年之约准备篇
主要地点: 加玛帝国帝都、炼药师公会、米特尔家族。 核心人物: 萧炎、药老、雅妃、海波东、法犸、古河、纳兰桀、纳兰嫣然。 萧炎状态: 大斗师至斗灵前后，炼药能力明显超越同龄人。 获得资源: 炼药师声望、丹药资源、人脉资源。 剧情意义: 这一篇让萧炎从“能打”变成“有身份、有资源、有号召力”的年轻强者。

主线事件：

1. 萧炎以炼药师身份进入帝都，参与炼药师大会或相关炼药剧情。

2. 借炼药能力获得资源、人脉与身份认可。

3. 解决纳兰桀烙毒等问题，与纳兰家族产生复杂联系。

4. 为三年之约积蓄实力。

云岚宗三年之约与云岚宗大战篇
主要地点: 云岚宗、加玛帝国。 核心人物: 萧炎、纳兰嫣然、云韵、云山、药老、海波东、加刑天、萧战、魂殿护法。 萧炎状态: 斗灵至斗王前后，依靠异火、药老与底牌可越级作战。 获得资源: 复仇目标升级，由个人退婚转为家族与魂殿之仇。 剧情意义: 三年之约结束了前期羞辱线，但萧战被抓开启了更大的魂殿线。

主线事件：

1. 萧炎赴约，与纳兰嫣然决战。

2. 三年之约完成，萧炎证明自身。

3. 云岚宗与萧家、魂殿的深层牵连暴露。

4. 萧战被魂殿势力掳走，萧炎与魂殿主线正式接轨。

5. 萧炎多次与云山、云岚宗冲突，最终离开加玛帝国前往黑角域。

关键战斗：萧炎 vs 纳兰嫣然；萧炎与云山、云岚宗冲突。
黑角域初入与迦南学院外院/内院篇
主要地点: 黑角域、迦南学院外院、内院。 核心人物: 萧炎、药老、萧玉、琥嘉、吴昊、薰儿、苏千、紫研、林修崖、柳擎、韩月、韩枫。 萧炎状态: 斗王前后，异火与炼药能力成为核心优势。 获得资源: 三千雷动、内院资源、陨落心炎线索、紫研羁绊。 剧情意义: 迦南学院篇是萧炎从地方强者迈向大陆舞台的第二个转折点。

主线事件：

1. 萧炎进入迦南学院，参与外院与内院选拔。

2. 内院火能制度、强榜制度、天焚炼气塔登场。

3. 萧炎建立磐门，与白帮、药帮等学生势力冲突。

4. 紫研登场，太虚古龙血脉线开始铺垫。

5. 韩枫作为药老叛徒与黑角域药皇身份登场。

陨落心炎暴动与韩枫冲突篇
主要地点: 天焚炼气塔、岩浆世界、黑角域。 核心人物: 萧炎、药老、苏千、紫研、韩枫、金银二老、黑角域强者。 萧炎状态: 斗王层次显著提升，二色异火体系成形。 获得资源: 陨落心炎、佛怒火莲威力跃升、岩浆世界线索。 剧情意义: 陨落心炎让萧炎正式拥有“修炼作弊器”，也使佛怒火莲成为中期代表底牌。

主线事件：

1. 天焚炼气塔中的陨落心炎暴动。

2. 韩枫联合黑角域强者觊觎陨落心炎。

3. 萧炎卷入地底岩浆世界，经历生死危机。

4. 萧炎获得陨落心炎，焚诀再次进化，实力大涨。

5. 药老因消耗过大或魂殿后续追捕，为被擒事件埋下伏笔。

关键战斗：萧炎/迦南学院 vs 韩枫及黑角域联军。
加玛帝国回归与炎盟建立篇

主线事件：

1. 萧炎回归加玛帝国，解决云岚宗旧账。

3. 炎盟建立，加玛帝国势力格局重组。

4. 萧炎不再只是复仇者，而开始具备组织整合能力。

中州初入与寻找药老线

主线事件：

1. 萧炎进入中州，发现自身过去的地方优势不再明显。

2. 小医仙厄难毒体线推进。

3. 天火尊者灵魂登场，提供五轮离火法等关键传承。

4. 风雷阁因三千雷动等问题与萧炎结怨。

5. 萧炎逐渐接触星陨阁、风尊者、药老旧友网络。

丹塔、丹会与三千焱炎火篇

主线事件：

2. 慕骨老人代表魂殿线搅局。

3. 萧炎获得丹会重要名次/冠军资格，进入星域接触三千焱炎火。

5. 三千焱炎火带来的星辰修复与不死特性，大幅增强萧炎续航。

关键战斗：丹会竞争；星域收火；与魂殿炼药师斗争。

远古八族冲突与魂族灭族清洗篇
主要地点: 古界、药界、炎界、雷界、魂界等。 核心人物: 萧炎、古元、魂天帝、虚无吞炎、炎烬、雷赢、药丹、药老、远古八族高层。 萧炎状态: 高阶斗圣潜力逐渐显露。 获得资源: 远古八族情报、药族/魂族矛盾、陀舍古帝玉终局线。 剧情意义: 主线彻底进入终局，魂族从“幕后黑手”变为公开灭世级敌人。

主线事件：

1. 魂族加速收集陀舍古帝玉。

2. 灵族、石族、药族等遭到魂族清洗或重创。

3. 药族药典等事件暴露魂族真正野心。

4. 古族、炎族、雷族、天府联盟被迫形成更高层级联军。

古帝洞府篇
主要地点: 迦南学院岩浆世界深处、陀舍古帝洞府。 核心人物: 萧炎、魂天帝、古元、烛坤、紫研、虚无吞炎、帝品雏丹、陀舍古帝残留意志。 萧炎状态: 九星斗圣前后，距离斗帝仅一步。 获得资源: 陀舍古帝传承、帝炎成形条件、斗帝秘密。 剧情意义: 古帝洞府是全书所有伏笔的收束点：异火、古玉、远古八族、斗帝断层全部在此汇合。

主线事件：

1. 陀舍古帝玉集齐，古帝洞府开启。

2. 烛坤被困洞府的真相揭开。

3. 帝品雏丹成为突破斗帝的关键资源。

4. 魂天帝夺取帝品雏丹，走向成帝。

5. 萧炎获得陀舍古帝传承与异火本源线索。

魂天帝成帝与中州浩劫篇
主要地点: 中州、魂族大阵、天府联盟阵地。 核心人物: 魂天帝、萧炎、古元、烛坤、天府联盟、远古族群。 萧炎状态: 九星斗圣巅峰，尚未真正成帝。 剧情意义: 这一阶段体现“斗圣与斗帝之间不可逾越的天堑”。

主线事件：

1. 魂天帝利用帝品雏丹与魂族万年布局突破斗帝。

2. 中州陷入浩劫，魂族大阵威胁全大陆生灵。

3. 古元、烛坤等九星斗圣也难以真正抗衡斗帝。

4. 萧炎必须以另一种方式获得斗帝力量。

萧炎成帝与双帝之战篇
主要地点: 中州战场、斗气大陆位面空间。 核心人物: 萧炎、魂天帝、古元、烛坤、薰儿、彩鳞、药老、天府联盟。 萧炎状态: 斗帝。 获得资源: 帝炎、炎帝血脉、位面巅峰权柄。 剧情意义: 个人逆袭、家族复兴、魂族阴谋、异火体系、远古斗帝传承全部完成终局闭环。

主线事件：

1. 萧炎融合异火本源，成就帝炎。

2. 萧炎突破斗帝，自号炎帝。

3. 炎帝萧炎与魂帝魂天帝展开双帝之战。

4. 萧炎以帝炎与众生之力封印魂天帝。

5. 萧族血脉复兴，斗气大陆进入新纪元。

尾声与大千世界线
主要地点: 斗气大陆、无尽火域、大千世界。 核心人物: 萧炎、萧薰儿、彩鳞、药老、萧族、天府联盟旧部。 剧情意义: 《斗破苍穹》本篇结束，大千世界线开启。

主线事件：

1. 萧炎成帝后重塑萧族血脉。

2. 与核心亲友前往更高位面。

3. 后续在《大主宰》中作为炎帝、无尽火域之主出现。

三、主线伏笔回收表

1. 戒指伏笔：药老寄居骨炎戒 → 萧炎重修 → 药老被擒 → 萧炎营救 → 师徒共同对抗魂族。 2. 退婚伏笔：纳兰嫣然退婚 → 三年之约 → 前期自证完成。 3. 异火伏笔：焚诀需要异火进化 → 青莲地心火 → 陨落心炎 → 三千焱炎火 → 骨灵冷火/海心焰等 → 净莲妖火 → 帝炎。 4. 萧族伏笔：萧家只是边陲小族 → 萧族为远古八族之一 → 萧玄失败导致血脉枯竭 → 萧炎成帝复兴萧族。 5. 古玉伏笔：萧家古玉不起眼 → 魂殿追查 → 八族古玉集齐 → 古帝洞府开启。 6. 迦南伏笔：天焚炼气塔底部岩浆世界 → 陀舍古帝洞府入口 → 终局回到迦南线。 7. 薰儿伏笔：萧薰儿身份神秘 → 古族神品血脉 → 远古八族主线。 8. 紫研伏笔：爱吃丹药的小女孩 → 太虚古龙皇族 → 古帝洞府烛坤线。 9. 魂殿伏笔：抓捕灵魂的黑暗势力 → 魂族白手套 → 魂天帝斗帝计划。 10. 丹药伏笔：炼药师地位极高 → 丹塔/丹会 → 帝品雏丹 → 斗帝突破。

1. 若用于写剧情复盘，建议以本时间线为骨架，再从“人物关系档案”“势力图谱”“秘境整理”中抽取细节。 2. 若用于战力讨论，必须结合“萧炎成长路线总表”和“关键战役复盘”，不能只看篇章顺序。 3. 若用于漫画解说，建议将每个阶段拆成“漫画名场面 + 原著补充 + 设定解释”三段式。
