## 斗气修为境界体系

| 境界 | 星级 | 核心能力 |
|------|------|---------|
| 斗之气 | 1-9段 | 入门，温养筋骨 |
| 斗者 | 1-9星 | 凝聚斗之气旋，可修炼功法 |
| 斗师 | 1-9星 | 斗气液态化，斗气纱衣 |
| 大斗师 | 1-9星 | 斗气固态化，斗气铠甲 |
| 斗灵 | 1-9星 | 斗气凝物化形 |
| 斗王 | 1-9星 | 斗气化翼，可飞行 |
| 斗皇 | 1-9星 | 大量调动天地能量 |
| 斗宗 | 1-9星 | 掌握空间之力，踏空而行 |
| 斗尊 | 1-9星(含九转) | 空间之力化境，可撕裂空间 |
| 半圣 | 初入→圆满 | 斗气向圣力转化 |
| 斗圣 | 1-9星 | 掌控空间法则，大陆顶级 |
| 斗帝 | 无星级 | 终极境界，天地法则化身 |

## 药尘（药老）核心资料

- **身份**：星陨阁阁主，萧炎师父
- **异火**：骨灵冷火（异火榜第11）
- **传承**：焚诀、炼药术、万兽鼎
- **经历**：药族出身 → 被逐 → 创建星陨阁 → 遭韩枫背叛 → 灵魂沉睡 → 收萧炎为徒 → 重生 → 斗圣

## 萧炎收服异火顺序

1. 青莲地心火（第19）— 塔戈尔大沙漠
2. 陨落心炎（第14）— 迦南学院
3. 三千焱炎火（第9）— 丹塔丹会
4. 骨灵冷火（第11）— 药老赠予
5. 海心焰（第15）— 慕骨老人
6. 净莲妖火（第3）— 妖火空间
7. 金帝焚天炎（第4）— 薰儿赠予
8. 虚无吞炎（第2）— 魂族
9. 其余异火 → 融合成帝炎（第1）
