《斗破苍穹》主线剧情时间线完整整理
资料口径：以漫画版主线呈现为优先；漫画中压缩、省略或尚未充分展开的设定，以原著小说逻辑补足。涉及境界、异火、斗技、战力判断时，仅作资料库整理，不把推演内容冒充为漫画明文。章节话数因平台版本、单行本、网络连载拆分可能存在差异，本文件按“剧情篇章顺序”组织。

============================================================
一、总览：主线推进的三条线
============================================================

1. 成长线
萧炎从乌坦城“斗之气三段”的废柴少年重新起步，在药老指导下重修斗气、学习炼药、吞噬异火、进化焚诀，最终由地方少年成长为斗气大陆顶峰的炎帝。

2. 家族线
萧炎的个人成长背后，是没落萧族与远古八族的旧怨。萧族因萧玄冲击斗帝失败而血脉枯竭，魂族暗中夺取陀舍古帝玉并清洗远古族群，萧炎最终以斗帝身份复兴萧族。

3. 反魂族线
前中期敌人看似是纳兰嫣然、云岚宗、韩枫、魂殿护法、各地强者；后期逐渐收束到魂殿与魂族。魂殿抓捕灵魂、追查古玉、谋夺异火和丹药，最终服务于魂天帝突破斗帝的大局。

二、主线时间线分篇整理
============================================================

【阶段01：乌坦城废柴与退婚篇】
主要地点：乌坦城、萧家、米特尔拍卖场。
核心人物：萧炎、萧战、萧薰儿、纳兰嫣然、药老、雅妃、萧宁、萧媚。
主线事件：
1. 萧炎少年时期曾是萧家天才，后因药老寄居戒指吸收斗气，境界连续倒退。
2. 纳兰嫣然随云岚宗来萧家退婚，萧炎立下“三年之约”。
3. 药老苏醒，解释萧炎斗气消失的原因，正式收萧炎为徒。
4. 萧炎重修，开始炼制筑基灵液，借米特尔拍卖场换取资源。
萧炎状态：斗之气低谷重新起步，尚未进入真正大陆舞台。
获得资源：药老、骨炎戒、筑基灵液、焚诀线索、炼药入门。
剧情意义：这是萧炎人格与主线目标的起点。“莫欺少年穷”不是单纯口号，而是他后续所有修炼、复仇、自证与守护的精神源头。
漫画表现重点：漫画通常强化退婚冲突、萧炎被羞辱后的爆发情绪、药老登场的转折感。

【阶段02：魔兽山脉修炼篇】
主要地点：魔兽山脉、青山镇、佣兵活动区域。
核心人物：萧炎、药老、小医仙、云韵、紫晶翼狮王、狼头佣兵团。
主线事件：
1. 萧炎离开乌坦城进入魔兽山脉修炼。
2. 结识小医仙，获得毒经、宝藏、飞行斗技等机缘。
3. 与云韵相遇，卷入紫晶翼狮王相关事件。
4. 通过实战磨炼，萧炎从家族少年转向真正的冒险者。
萧炎状态：斗者至斗师阶段快速成长。
获得资源：飞行斗技线索、药材、毒经关联、小医仙羁绊、云韵羁绊。
剧情意义：这一篇完成萧炎从“家族子弟”到“独立修炼者”的身份转换，也埋下小医仙、云韵两条重要人物线。

【阶段03：塔戈尔沙漠与青莲地心火篇】
主要地点：塔戈尔大沙漠、石漠城、蛇人族领地。
核心人物：萧炎、药老、美杜莎女王/彩鳞、青鳞、海波东、萧鼎、萧厉、古河、云韵。
主线事件：
1. 萧炎前往塔戈尔沙漠寻找异火。
2. 与大哥萧鼎、二哥萧厉重逢。
3. 结识青鳞，青鳞的碧蛇三花瞳设定出现。
4. 海波东登场，破除封印成为萧炎早期重要盟友。
5. 蛇人族、美杜莎女王、古河一行围绕青莲地心火发生争夺。
6. 美杜莎借青莲地心火进化，意外形成七彩吞天蟒状态。
7. 萧炎最终取得青莲地心火，并推动焚诀第一次核心进化。
萧炎状态：大斗师阶段，初步掌握异火力量。
获得资源：青莲地心火、海波东盟友、七彩吞天蟒/彩鳞羁绊、青鳞人物线。
关键战斗：沙漠异火争夺、美杜莎进化、萧炎逃生。
剧情意义：青莲地心火是萧炎真正意义上的第一朵异火，标志焚诀从潜力功法转入可成长轨道。

【阶段04：帝都炼药与三年之约准备篇】
主要地点：加玛帝国帝都、炼药师公会、米特尔家族。
核心人物：萧炎、药老、雅妃、海波东、法犸、古河、纳兰桀、纳兰嫣然。
主线事件：
1. 萧炎以炼药师身份进入帝都，参与炼药师大会或相关炼药剧情。
2. 借炼药能力获得资源、人脉与身份认可。
3. 解决纳兰桀烙毒等问题，与纳兰家族产生复杂联系。
4. 为三年之约积蓄实力。
萧炎状态：大斗师至斗灵前后，炼药能力明显超越同龄人。
获得资源：炼药师声望、丹药资源、人脉资源。
剧情意义：这一篇让萧炎从“能打”变成“有身份、有资源、有号召力”的年轻强者。

【阶段05：云岚宗三年之约与云岚宗大战篇】
主要地点：云岚宗、加玛帝国。
核心人物：萧炎、纳兰嫣然、云韵、云山、药老、海波东、加刑天、萧战、魂殿护法。
主线事件：
1. 萧炎赴约，与纳兰嫣然决战。
2. 三年之约完成，萧炎证明自身。
3. 云岚宗与萧家、魂殿的深层牵连暴露。
4. 萧战被魂殿势力掳走，萧炎与魂殿主线正式接轨。
5. 萧炎多次与云山、云岚宗冲突，最终离开加玛帝国前往黑角域。
萧炎状态：斗灵至斗王前后，依靠异火、药老与底牌可越级作战。
获得资源：复仇目标升级，由个人退婚转为家族与魂殿之仇。
关键战斗：萧炎 vs 纳兰嫣然；萧炎与云山、云岚宗冲突。
剧情意义：三年之约结束了前期羞辱线，但萧战被抓开启了更大的魂殿线。

【阶段06：黑角域初入与迦南学院外院/内院篇】
主要地点：黑角域、迦南学院外院、内院。
核心人物：萧炎、药老、萧玉、琥嘉、吴昊、薰儿、苏千、紫研、林修崖、柳擎、韩月、韩枫。
主线事件：
1. 萧炎进入迦南学院，参与外院与内院选拔。
2. 内院火能制度、强榜制度、天焚炼气塔登场。
3. 萧炎建立磐门，与白帮、药帮等学生势力冲突。
4. 紫研登场，太虚古龙血脉线开始铺垫。
5. 韩枫作为药老叛徒与黑角域药皇身份登场。
萧炎状态：斗王前后，异火与炼药能力成为核心优势。
获得资源：三千雷动、内院资源、陨落心炎线索、紫研羁绊。
剧情意义：迦南学院篇是萧炎从地方强者迈向大陆舞台的第二个转折点。

【阶段07：陨落心炎暴动与韩枫冲突篇】
主要地点：天焚炼气塔、岩浆世界、黑角域。
核心人物：萧炎、药老、苏千、紫研、韩枫、金银二老、黑角域强者。
主线事件：
1. 天焚炼气塔中的陨落心炎暴动。
2. 韩枫联合黑角域强者觊觎陨落心炎。
3. 萧炎卷入地底岩浆世界，经历生死危机。
4. 萧炎获得陨落心炎，焚诀再次进化，实力大涨。
5. 药老因消耗过大或魂殿后续追捕，为被擒事件埋下伏笔。
萧炎状态：斗王层次显著提升，二色异火体系成形。
获得资源：陨落心炎、佛怒火莲威力跃升、岩浆世界线索。
关键战斗：萧炎/迦南学院 vs 韩枫及黑角域联军。
剧情意义：陨落心炎让萧炎正式拥有“修炼作弊器”，也使佛怒火莲成为中期代表底牌。

【阶段08：加玛帝国回归与炎盟建立篇】
主要地点：加玛帝国、云岚宗、萧家故地。
核心人物：萧炎、海波东、萧鼎、萧厉、雅妃、加刑天、云韵、云山、魂殿护法。
主线事件：
1. 萧炎回归加玛帝国，解决云岚宗旧账。
2. 云岚宗覆灭，云韵与萧炎关系进入遗憾线。
3. 炎盟建立，加玛帝国势力格局重组。
4. 萧炎不再只是复仇者，而开始具备组织整合能力。
萧炎状态：斗王至斗皇前后，依靠异火可抗衡更高境界。
获得资源：炎盟根基、加玛帝国资源、人脉整合。
剧情意义：这一篇完成前期地图的收束，为萧炎离开西北大陆进入中州做准备。

【阶段09：中州初入与寻找药老线】
主要地点：中州、空间虫洞、各大城池、风雷阁相关区域。
核心人物：萧炎、小医仙、天火尊者、风尊者、凤清儿、风雷阁强者、魂殿尊老。
主线事件：
1. 萧炎进入中州，发现自身过去的地方优势不再明显。
2. 小医仙厄难毒体线推进。
3. 天火尊者灵魂登场，提供五轮离火法等关键传承。
4. 风雷阁因三千雷动等问题与萧炎结怨。
5. 萧炎逐渐接触星陨阁、风尊者、药老旧友网络。
萧炎状态：斗皇至斗宗过渡，灵魂与炼药能力继续成长。
获得资源：五轮离火法、天火尊者、风尊者线索、星陨阁人脉。
剧情意义：中州篇使作品从“区域冒险”升级为“大陆强者体系”。

【阶段10：丹塔、丹会与三千焱炎火篇】
主要地点：圣丹城、丹塔、丹界、星域。
核心人物：萧炎、曹颖、丹晨、宋清、玄空子、慕骨老人、三千焱炎火、紫研相关龙印线。
主线事件：
1. 萧炎参加丹会，展示顶级炼药天赋。
2. 慕骨老人代表魂殿线搅局。
3. 萧炎获得丹会重要名次/冠军资格，进入星域接触三千焱炎火。
4. 借助太虚古龙相关力量，萧炎收服三千焱炎火。
5. 三千焱炎火带来的星辰修复与不死特性，大幅增强萧炎续航。
萧炎状态：斗宗巅峰至斗尊前后，炼药进入八品体系。
获得资源：三千焱炎火、丹塔认可、星域经验、炼药师地位。
关键战斗：丹会竞争；星域收火；与魂殿炼药师斗争。
剧情意义：萧炎由“年轻强者”进入中州顶级天才行列。

【阶段11：营救药老与星陨阁重整篇】
主要地点：亡魂山脉、魂殿分殿、星陨阁。
核心人物：萧炎、风尊者、天火尊者、小医仙、药老、摘星老鬼、魂殿尊老。
主线事件：
1. 萧炎探查药老被囚位置。
2. 组织强者营救药老，与魂殿发生正面冲突。
3. 药老重塑肉身或逐步恢复。
4. 星陨阁成为萧炎在中州的核心根据地。
萧炎状态：斗尊级别战力逐步成形。
获得资源：药老回归、星陨阁根基、对抗魂殿的组织平台。
关键战斗：亡魂山脉/魂殿分殿营救战。
剧情意义：师徒线完成重大回收，萧炎正式从被药老保护者转变为能救回老师的强者。

【阶段12：四方阁、花宗、焚炎谷与中州势力整合篇】
主要地点：星陨阁、花宗、焚炎谷、风雷阁势力范围。
核心人物：萧炎、药老、风尊者、唐震、唐火儿、云韵、花宗强者、凤清儿、风雷阁强者。
主线事件：
1. 萧炎与焚炎谷接触，九龙雷罡火线出现。
2. 云韵在花宗线中再次出现，情感线与势力线交错。
3. 风雷阁、天妖凰族等与萧炎矛盾加深。
4. 星陨阁、丹塔、焚炎谷、花宗等逐渐构成反魂殿联盟基础。
萧炎状态：斗尊阶段，斗技与异火体系更成熟。
获得资源：焚炎谷人脉、花宗盟友、九龙雷罡火相关信息。
剧情意义：这一篇不是单纯支线，而是天府联盟形成前的势力整合阶段。

【阶段13：古族与天墓篇】
主要地点：古界、古族、天墓。
核心人物：萧炎、萧薰儿、古元、古青阳、古妖、萧玄残魂、远古八族天才。
主线事件：
1. 萧炎进入古族，面对远古种族的血脉等级与身份壁垒。
2. 萧薰儿的古族神品血脉与身份正式展开。
3. 天墓历练中，萧炎见到萧玄残魂。
4. 萧玄传承帮助萧炎理解萧族兴衰，并增强萧族血脉与斗技基础。
萧炎状态：斗尊至半圣前后，血脉线正式觉醒。

关键战斗：古族年轻一代冲突、天墓历练。
剧情意义：萧炎从“萧家少年”回归“远古萧族后裔”，主线层级提升到远古八族。

【阶段14：莽荒古域与菩提古树篇】
主要地点：莽荒古域、菩提古树空间。
核心人物：萧炎、彩鳞、小医仙、紫研、各族强者、天妖凰族、远古魔兽。
主线事件：
1. 大陆强者进入莽荒古域争夺菩提古树机缘。

萧炎状态：半圣至斗圣门槛。
获得资源：菩提心/菩提子、圣境突破契机、心境升华。
剧情意义：菩提古树是萧炎从斗尊/半圣跨向斗圣的关键跳板。

【阶段15：净莲妖火篇】
主要地点：妖火空间。
核心人物：萧炎、净莲妖火、净莲妖圣残影、魂殿/魂族强者、古族/炎族/雷族强者。

1. 净莲妖火现世，引发中州顶级势力争夺。

4. 萧炎最终收服净莲妖火，异火体系发生质变。

获得资源：净莲妖火、梦魇天雾/净化能力、毁灭火莲进一步提升。

剧情意义：净莲妖火是萧炎成帝前最重要的异火之一，代表其从中州强者进入大陆巅峰序列。

【阶段16：天府联盟成立与魂殿战争篇】
主要地点：星陨阁、丹塔、花宗、焚炎谷、魂殿分殿。
核心人物：萧炎、药老、风尊者、丹塔三巨头、唐震、云韵、魂灭生、魂殿天尊。
主线事件：
1. 魂殿威胁全面升级，单一宗门难以应对。
2. 萧炎与药老牵头整合星陨阁、丹塔、焚炎谷、花宗等力量。
3. 天府联盟形成，与魂殿展开正面战争。
4. 魂殿多处分殿被攻破，魂族逐渐从幕后走向台前。
萧炎状态：斗圣级核心战力，兼具炼药、异火、组织号召力。
获得资源：天府联盟、反魂殿军事平台。
关键战斗：星陨阁大战、人殿/地殿/魂殿分殿战。
剧情意义：这是萧炎从个人强者转型为大陆联盟领袖的阶段。

【阶段17：远古八族冲突与魂族灭族清洗篇】
主要地点：古界、药界、炎界、雷界、魂界等。
核心人物：萧炎、古元、魂天帝、虚无吞炎、炎烬、雷赢、药丹、药老、远古八族高层。
主线事件：
1. 魂族加速收集陀舍古帝玉。
2. 灵族、石族、药族等遭到魂族清洗或重创。
3. 药族药典等事件暴露魂族真正野心。
4. 古族、炎族、雷族、天府联盟被迫形成更高层级联军。
萧炎状态：高阶斗圣潜力逐渐显露。
获得资源：远古八族情报、药族/魂族矛盾、陀舍古帝玉终局线。
剧情意义：主线彻底进入终局，魂族从“幕后黑手”变为公开灭世级敌人。

【阶段18：古帝洞府篇】
主要地点：迦南学院岩浆世界深处、陀舍古帝洞府。
核心人物：萧炎、魂天帝、古元、烛坤、紫研、虚无吞炎、帝品雏丹、陀舍古帝残留意志。
主线事件：
1. 陀舍古帝玉集齐，古帝洞府开启。
2. 烛坤被困洞府的真相揭开。
3. 帝品雏丹成为突破斗帝的关键资源。
4. 魂天帝夺取帝品雏丹，走向成帝。
5. 萧炎获得陀舍古帝传承与异火本源线索。
萧炎状态：九星斗圣前后，距离斗帝仅一步。
获得资源：陀舍古帝传承、帝炎成形条件、斗帝秘密。
剧情意义：古帝洞府是全书所有伏笔的收束点：异火、古玉、远古八族、斗帝断层全部在此汇合。

【阶段19：魂天帝成帝与中州浩劫篇】
主要地点：中州、魂族大阵、天府联盟阵地。
核心人物：魂天帝、萧炎、古元、烛坤、天府联盟、远古族群。
主线事件：
1. 魂天帝利用帝品雏丹与魂族万年布局突破斗帝。
2. 中州陷入浩劫，魂族大阵威胁全大陆生灵。
3. 古元、烛坤等九星斗圣也难以真正抗衡斗帝。
4. 萧炎必须以另一种方式获得斗帝力量。
萧炎状态：九星斗圣巅峰，尚未真正成帝。
剧情意义：这一阶段体现“斗圣与斗帝之间不可逾越的天堑”。

【阶段20：萧炎成帝与双帝之战篇】
主要地点：中州战场、斗气大陆位面空间。
核心人物：萧炎、魂天帝、古元、烛坤、薰儿、彩鳞、药老、天府联盟。
主线事件：
1. 萧炎融合异火本源，成就帝炎。
2. 萧炎突破斗帝，自号炎帝。
3. 炎帝萧炎与魂帝魂天帝展开双帝之战。
4. 萧炎以帝炎与众生之力封印魂天帝。
5. 萧族血脉复兴，斗气大陆进入新纪元。
萧炎状态：斗帝。
获得资源：帝炎、炎帝血脉、位面巅峰权柄。
剧情意义：个人逆袭、家族复兴、魂族阴谋、异火体系、远古斗帝传承全部完成终局闭环。

【阶段21：尾声与大千世界线】
主要地点：斗气大陆、无尽火域、大千世界。
核心人物：萧炎、萧薰儿、彩鳞、药老、萧族、天府联盟旧部。
主线事件：
1. 萧炎成帝后重塑萧族血脉。
2. 与核心亲友前往更高位面。
3. 后续在《大主宰》中作为炎帝、无尽火域之主出现。
剧情意义：《斗破苍穹》本篇结束，大千世界线开启。

三、主线伏笔回收表
============================================================

1. 戒指伏笔：药老寄居骨炎戒 → 萧炎重修 → 药老被擒 → 萧炎营救 → 师徒共同对抗魂族。
2. 退婚伏笔：纳兰嫣然退婚 → 三年之约 → 前期自证完成。
3. 异火伏笔：焚诀需要异火进化 → 青莲地心火 → 陨落心炎 → 三千焱炎火 → 骨灵冷火/海心焰等 → 净莲妖火 → 帝炎。
4. 萧族伏笔：萧家只是边陲小族 → 萧族为远古八族之一 → 萧玄失败导致血脉枯竭 → 萧炎成帝复兴萧族。
5. 古玉伏笔：萧家古玉不起眼 → 魂殿追查 → 八族古玉集齐 → 古帝洞府开启。
6. 迦南伏笔：天焚炼气塔底部岩浆世界 → 陀舍古帝洞府入口 → 终局回到迦南线。
7. 薰儿伏笔：萧薰儿身份神秘 → 古族神品血脉 → 远古八族主线。
8. 紫研伏笔：爱吃丹药的小女孩 → 太虚古龙皇族 → 古帝洞府烛坤线。
9. 魂殿伏笔：抓捕灵魂的黑暗势力 → 魂族白手套 → 魂天帝斗帝计划。
10. 丹药伏笔：炼药师地位极高 → 丹塔/丹会 → 帝品雏丹 → 斗帝突破。
